## 🎯 What It Does

Context Bridge maintains persistent AI context across BlackRoad OS, Claude, GitHub Copilot, and Google Gemini.

**Problem Solved**:
- No more re-explaining your project context in every new conversation
- Maintain consistent context across different AI platforms
- Update once, propagate everywhere

**How It Works**:
1. Store your context in a GitHub Gist
2. Install Context Bridge extension or CLI
3. Configure with your Gist URL
4. One-click context insertion in any AI chat

---

## 🚀 Quick Start

### CLI Usage
```bash
# Initialize with a template
context init developer

# View your context
context view

# Update your context
context update

# Get your Gist URL
context url
```

### Browser Extension Usage
1. Load extension in Chrome/Firefox
2. Visit Claude, BlackRoad OS, Copilot, or Gemini
3. Click "Insert Context" button in chat
4. Your context appears instantly
